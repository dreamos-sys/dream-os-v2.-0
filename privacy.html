1. Introduction

Welcome to Dream OS v2.0 ("we", "our", or "us"). We are committed to protecting your privacy and ensuring the security of your personal information. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our Dream OS application (the "App").

Dream OS is an integrated management system with spiritual wellness features, including prayer time tracking, module management (Booking, K3, Security, Maintenance, etc.), and productivity tools. By using the App, you consent to the data practices described in this policy.

🕌 Our Mission: "The Power Soul of Shalawat" — We integrate spiritual wellness with enterprise productivity while maintaining the highest standards of data privacy and security.

2. Data We Collect

2.1 Information You Provide

  • Access Credentials: Access key (hashed with SHA-256 + pepper) for authentication
  • User Profile: Name, role (Developer, Master, Admin, Security, Janitor, etc.), permissions
  • Form Submissions: Booking requests, K3 reports, maintenance requests, security logs
  • Preferences: Language selection (AR/EN/ID), theme preference (light/dark mode)

2.2 Information Collected Automatically

  • Device Information: Browser type, OS version, device model, screen resolution
  • Usage Data: Module access logs, feature usage frequency, session duration (anonymized)
  • Location Data: General location (city-level only, e.g., "Depok") for prayer time & weather features
  • Network Information: IP address (for security & rate limiting), connection type
  • Battery Status: Battery level (for UX optimization, not stored)

2.3 Information from Third Parties

  • Weather Data: From OpenWeatherMap API (via AllOrigins proxy)
  • Prayer Times: Calculated locally based on location timezone
  • Supabase: Database service for data storage (hosted in Indonesia/Singapore region)
⚠️ We do NOT collect: Sensitive personal data (religion, health records, biometric data), precise GPS location, contacts, call logs, SMS, camera/microphone access (unless explicitly granted for future features).

3. How We Use Your Data

We use the collected information for the following purposes:

  • ✅ Authentication: Verify user identity and manage access permissions
  • ✅ Core Functionality: Enable booking, K3 reporting, security logs, maintenance requests
  • ✅ Personalization: Customize UI based on role, language, theme preferences
  • ✅ Spiritual Features: Display prayer times, change background based on prayer schedule, show shalawat reminders
  • ✅ Performance: Optimize app performance based on device tier (high/mid/low)
  • ✅ Security: Detect and prevent fraud, abuse, rate limiting (5 attempts, 15min lockout)
  • ✅ Analytics: Understand usage patterns to improve features (anonymized, aggregated)
  • ✅ Communication: Send push notifications for booking confirmations, prayer reminders (if enabled)
  • ✅ Compliance: Meet legal obligations under Indonesian PDP Law & GDPR

5. Data Retention

We retain personal information only for as long as necessary to fulfill the purposes outlined in this policy:

  • Session Data: 24 hours (stored in sessionStorage, cleared on logout)
  • Authentication Logs: 90 days (anonymized for security analysis)
  • Booking Records: 2 years (or per enterprise client agreement)
  • K3 Reports: 5 years (compliance with Indonesian labor law)
  • Usage Analytics: 180 days (aggregated, anonymized)
  • Deleted Accounts: Data permanently deleted within 30 days of request

After retention period, data is either anonymized, aggregated, or securely deleted.

6. Security Measures

We implement industry-standard security measures to protect your information:

  • 🔐 Encryption in Transit: HTTPS/TLS 1.3 for all data transmission
  • 🔐 Encryption at Rest: Supabase database encryption (AES-256)
  • 🔐 Password Hashing: SHA-256 with pepper (server-side secret)
  • 🔐 Rate Limiting: 5 failed attempts → 15-minute lockout
  • 🔐 Content Security Policy: Strict CSP headers to prevent XSS
  • 🔐 X-Frame-Options: DENY to prevent clickjacking
  • 🔐 Service Worker: Offline-first with secure caching strategies
  • 🔐 Access Control: Role-based permissions (Developer, Master, Admin, etc.)
  • 🔐 Audit Logs: All actions logged for accountability
🏆 ISO 27001 Alignment: Our security controls align with ISO 27001 Annex A requirements for enterprise clients.

7. Your Rights (GDPR/PDP)

Under Indonesian PDP Law (UU PDP No. 27/2022) and GDPR, you have the following rights:

  • ✅ Right to Access: Request copy of your personal data
  • ✅ Right to Correction: Request correction of inaccurate data
  • ✅ Right to Deletion: Request deletion of your data ("right to be forgotten")
  • ✅ Right to Portability: Receive your data in machine-readable format
  • ✅ Right to Restriction: Request limitation of data processing
  • ✅ Right to Object: Object to certain data processing activities
  • ✅ Right to Withdraw Consent: Withdraw consent at any time

To exercise these rights, contact us at privacy@dreamos.id. We will respond within 30 days.

8. Cookies & Local Storage

We use the following local storage technologies:

  • localStorage: User preferences (theme, language), session persistence (encrypted)
  • sessionStorage: Temporary session data (cleared on browser close)
  • Service Worker Cache: Offline functionality (cache versioned, auto-cleaned)
  • IndexedDB: Offline form submissions (booking, K3 reports)

No third-party cookies are used. All storage is first-party and essential for app functionality.

9. Children's Privacy
